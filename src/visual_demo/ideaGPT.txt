## 具体绘制步骤

1. **共用准备**

   ```python
   import pandas as pd, matplotlib.pyplot as plt, seaborn as sns
   COLOR = {'A':'#1f77b4','B':'#ff7f0e','C':'#2ca02c'}   # 固定色盘
   ```

2. **① 对照散点 + 箭头**

   ```python
   fig, ax = plt.subplots(1,2,figsize=(8,3.5), sharex=True, sharey=True)
   for i,(df,title) in enumerate([(dirty,'Dirty'), (clean,'Cleaned')]):
       for seg,c in COLOR.items():
           sub = df[df.segment==seg]
           ax[i].scatter(sub.age, sub.income, c=c, s=20, alpha=.6, label=seg if i==0 else "")
       ax[i].set_title(title); ax[i].grid(ls=':')
   # 只在左图加箭头
   for idx in moved_idx:          # moved_idx = df.diff_row_index
       x0,y0 = dirty.loc[idx,['age','income']]
       x1,y1 = clean.loc[idx,['age','income']]
       ax[0].arrow(x0,y0, x1-x0, y1-y0, width=0.05, head_width=1, color='k', alpha=.4)
   plt.legend(); plt.tight_layout(); plt.savefig('scatter_compare.pdf')
   ```

3. **② 过程曲线（SSE vs. 迭代步）**

   ```python
   plt.plot(range(len(sse_dirty)), sse_dirty, 'o-', label='Dirty')
   plt.plot(range(len(sse_clean)), sse_clean, 's--', label='Cleaned')
   plt.annotate("steps↓ 43%", xy=(len(sse_clean)-1, sse_clean[-1]),
                xytext=(.6,.8), textcoords='axes fraction',
                arrowprops=dict(arrowstyle='->'))
   plt.ylabel('SSE'); plt.xlabel('Iteration'); plt.legend(); plt.tight_layout()
   ```
